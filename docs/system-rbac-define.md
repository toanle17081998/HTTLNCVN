### Role Capabilities & Access Control

* **Guest**
    * **Public Access:** View landing page, "About Us," and "Contact" information.
    * **Restricted Content:** View only blog posts tagged as "Public."
    * **Interaction:** No ability to comment, enroll, or access internal prayer journals.

* **Church Member**
    * **Prayer Journal:** Create, edit, and delete personal prayers. Toggle prayer visibility between "Private" (self) and "Church-wide."
    * **Learning:** Enroll in courses, access lessons, take quizzes/exams, and view earned certificates.
    * **Engagement:** View all public and internal-only blog posts and community events.
    * **Search:** Access personalized search results and recommendations based on activity.

* **Church Admin**
    * **Content Management:** Create and manage all blog posts (Public/Private), "About Us" content, and contact details.
    * **LMS Management:** Create courses, lessons, and tests; enroll members into specific learning paths.
    * **Operations:** Add events to the calendar and configure automated Telegram notifications.
    * **Moderation:** Oversee shared prayer journals and manage local church member permissions.

* **System Admin**
    * **Infrastructure:** Configure global Role/Permission schemas and system-wide settings.
    * **Integrations:** Manage Telegram API webhooks, notification servers, and database configurations.
    * **Oversight:** Full access to all data, including prayer logs, quiz results, and user analytics.
    * **Maintenance:** Execute system updates, backups, and security audits.
